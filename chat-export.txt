notes from friday - Answer generated by read.ai:

TL;DR

Core goal:
Build a “Coordination Games / Olympiad” for AI agents — but start simple, iterate fast, and only scale complexity once it’s clearly working.

🧠 Key Decisions & Direction

1. Start simple → add depth later
• Strong consensus: use simple game mechanics (Prisoner’s Dilemma, Stag Hunt, etc.)
• Add depth via:
• iteration over many rounds
• memory / reputation (trust over time)
• multi-game dynamics
• Framing: simple rules + deep emergent behavior

2. Economic model = early + unfinished
• Rough budget: see https://docs.google.com/spreadsheets/d/17sonofcNxnFHbR6Tyst8xtwMHxojxJf-rogcqBzZaQQ/edit?gid=0#gid=0
• Structure:
• rehearsals → dress rehearsals → main event
• small buy-in (~$5), platform fee (~20%)
• Open questions:
• token mechanics (mint/burn, supply)
• fee structure (per game vs entry)
• business model still very TBD

3. Timeline (aggressive, flexible)
• Rehearsals starting late April
• Multiple dress rehearsals → main event late May
• No hard commitment until it “feels amazing”

4. Games shortlist (lean set, high quality)
Likely initial set:
• Iterated Prisoner’s Dilemma
• Tragedy of the Commons
• Schelling Point
• Capture the Flag (team coordination)
• Possibly Stag Hunt

Bias: fewer, better games

5. UX / Spectator insight (important)
• Raw agent interactions are boring (infinite backrooms lesson)
• Need:
• highlights, not full streams
• aggregation over many games
• storytelling layer (key moments, anomalies)

6. Aesthetics
• Likely:
• shared design system (Octavian)
• but allow “pocket universes” per game
• Goal: cohesion + flexibility

⚡ Key Open Problems
• Reward mechanics + token design
• Business model (fees, external markets like Polymarket)
• Game complexity boundary (simple vs rich)
• How to visualize trust / reasoning / strategy
• Who owns the financial modeling role

🧭 Meta Insight

The real product isn’t just games — it’s: a benchmark + data layer for agent coordination across many games over time That’s where the long-term value + research angle is.
